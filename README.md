# Single Cell RNA-seq Analysis

This repository contains analysis performed for the Single Cell Bioinformatics course at Saarland University.

## Tools Used
- R
- Seurat
- SingleR
- DoubletFinder
- tidyverse

## Analysis Workflow
1. Data preprocessing and quality control
2. Filtering low-quality cells
3. Doublet detection
4. Normalization and feature selection
5. PCA and UMAP dimensionality reduction
6. Clustering of cell populations
7. Cell type annotation
8. Differential expression analysis

## Dataset
Public scRNA-seq dataset of immune cells from COVID-19 patients.
